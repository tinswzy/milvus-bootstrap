Milvus 依赖的各组件服务实例，按类型分组、点击展开看每个实例。集群内工作负载（etcd / MinIO / MQ）显示运行的 image tag 与可升级版本；外部对象存储（S3 / Blob / OSS / COS / OBS）显示接入点、无版本升级。每个分类右上角可创建该类型的新实例。Milvus 实例本身在「Milvus Instances」中管理。

两类依赖实例：集群内工作负载（Strimzi / 各 operator 管理）独立升级、互不影响；外部对象存储是托管接入端点（凭证 + endpoint），无法"升级"，只能创建 / 编辑 / 解绑。Milvus 实例在创建或切换时从这些依赖实例里选用。
